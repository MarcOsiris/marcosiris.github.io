Marcie Tech Tips

Welkom bij Marcie Tech Tips! Dit is een vervanging van #marcie-tips op discord, zie #techsupport.

Dit zijn vooral tips van programma's en apps. Deze zijn als suggesties bedoelt en zijn mijn mening, dus niet haten als je het er niet mee eens bent. Bijna elk programma is FOSS (Free and Open Source Software), dit betekent dat je naar de code kan kijken (dit gebeurt meestal op github of gitlab). FOSS is gratis en zeer betrouwbaar, omdat je alles kan zien wat het doet en installeert. Chrome, Discord, Windows etc. zijn closed source, dus je kan niet naar de code kijken. Dit kan impact hebben op je privacy en security, doordat ze bijvoorbeeld je gegevens door verkopen voor geld. FOSS heeft dit probleem niet. Ik ga ook praten over forks, dit zijn programma's die de code van een ander programma gebruikt hebben en er op zijn gebaseert. Security is hoe goed je beveiligt bent, meestal gebeurt dit met een firewall of iets dergelijks. Dit is iets ANDERS dan privacy, je weet vast wel wat privacy is.
