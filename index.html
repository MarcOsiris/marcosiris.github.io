Welkom bij Marcie Tech Tips!

Dit is een vervanging van #marcie-tips op Discord, zie #techsupport.

Dit zijn vooral tips voor programma's en apps. Deze zijn als suggesties bedoelt en zijn mijn mening, dus niet haten als je het er niet mee eens bent. Bijna elk programma is FOSS (Free and Open Source Software), dit betekent dat je naar de source code kan kijken (dit gebeurt meestal op Github of Gitlab). FOSS is gratis en zeer betrouwbaar, omdat je alles kan zien wat het doet en installeert. Chrome, Discord, Windows etc. zijn closed source, dus je kan niet naar de source code kijken. Dit kan impact hebben op je privacy en security, doordat ze bijvoorbeeld je gegevens door verkopen voor geld. FOSS heeft dit probleem niet.

Ik ga ook praten over forks, dit zijn programma's die de source code van een ander programma gebruikt hebben en er op zijn gebaseert.

Security is hoe goed je beveiligt bent voor 4russen bijvoorbeeld. Dit is iets anders dan privacy, je weet vast wel wat privacy is.

Laatst werd ook gevraagt of de website ZELF wel open source is... Natuurlijk is het dat! Je kan zelf naar de source code kijken op Github.

Waarom ben ik hier aan begonnen?

Ik wilde graag mijn kennis delen met andere mensen, zodat ook zij meer te weten komen over programma's en computers in het algemeen. Ik had dit eerst op Discord staan, maar ik kon daar heel weinig mee. Daarom heb ik besloten om een eigen website te maken hiervoor!

Problemen met de site?

We doen heel erg ons best om de site zo goed mogelijk te maken voor iedereen. Als niet alles op je scherm past, kan je of je resolutie veranderen (in je Windows settings), of je kan in je browser in/uitzoomen (ctrl inhouden + scrollen naar beneden of het aanpassing in je settings).

De site is niet officieel bedoelt voor telefoon en er zijn veel problemen mee. Echter werkt het wel gewoon.
