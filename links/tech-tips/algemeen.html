Algemeen

Dit zijn algemene tips om je PC (Personal Computer) sneller en opgeruimder te maken.

Performance boost

Dit zijn een paar tips om je pc sneller te maken:

  • Verwijder apps die je niet meer gebruikt

  • Sluit apps af die in de achtergrond runnen

  • Ruim je zooi op (files, downloads etc.)

  • Leer een beetje organiseren met mappies

  • Zet startup apps uit (dit kan in Windows settings of via task manager (ctrl + shift + esc))

  • Gebruik Windows ingebouwde opschoner, genaamt "Disk Clean-up"

    • (CCleaner is "spyware" (zie LTT zijn video) en slecht voor privacy. BleachBit is ook niet zo goede optie, omdat dit slecht is voor je disk en het brengt veel problemen met zich mee.)

  • Pas Windows instellingen aan voor betere snelheid

  • Gebruik niet meerdere VPN's tegelijk (VPN's maken je PC zowiezo langzamer)

  • Anti-4russen maken je PC ook langzamer, gebruik gewoon Windows ingebouwde anti-4rus

  • Sluit de ene game af voordat je de andere gaat spelen

  • Verander je settings in-game (V-Sync aan/uit, resolutie, graphics, anti-aliasing, render distance, full screen of borderless

    •  

    (Alleen als je ervaren bent!):

  • Verander je settings in je BIOS/update je BIOS

  • Zie "Terminal" voor een script om Windows te debloaten

  • Verander je OS

  • Gebruik MSI Afterburner (closed source) om te overclocken

    •  

    Veel mensen noemen dit ook wel "bloat" en het verminderen kan een enorme performance boost geven.

    Up-to-date blijven

    Je systeem updaten kan op verschillende manieren:

  • Windows updates

  • Microsoft Store (ja, dit staat los van Windows updates)

  • Winget, Chocolatey en Scoop (winget upgrade --all, choco upgrade all en scoop upgrade all)

  • In-app updates (browser, Discord, Steam etc.)

    •  

    Drivers:

  • AMD Radeon Software

  • Nvidia Geforce

  • Intel Driver & Support Assistant (voor Dell laptops: SupportAssist)

    • Graphics APIs

    In emulators als Dolphin, Retroarch, Citra, Ryujinx etc. zal ik aanraden om Vulkan te gebruiken. Vulkan is een manier om je graphics te renderen. Je hebt ook OpenGL en DirectX (11 en 12 zijn de nieuwste). Vulkan is meestal de snelste en is cross platform (dit houdt in dat het op meerdere systemen te gebruiken is, zoals Linux, Android en Windows (zie "OS's")), maar DirectX is zeer zeker geen slechte keuze. DirectX is alleen voor Windows. AMD heeft geen support voor OpenGL (nou ja, is er een aantal jaren geleden mee gestopt) en is heel langzaam. Minecraft bijvoorbeeld gebruikt wel nog steeds OpenGL, daarom is een Nvidia graphics card vaak ook een stuk beter voor Minecraft. Voor AMD kan je op Linux spelen. Linux heeft iets genaamt Mesa en dit heeft wel support voor OpenGL. Dus Minecraft is ook een stuk sneller op Linux!ðŸ¥°

    In het kort: Vulkan > DirectX (hoe nieuwer, hoe beter, maar langzamer) > OpenGL
