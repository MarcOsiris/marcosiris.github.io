Wat is een OS?

OS staat voor Operating System en bestuurt je hardware. Je hebt veel verschillende OS's: Windows, Linux, MacOS, iOS en BSD. De rest is gewoon gebaseert op een van die OS's. Bijvoorbeeld Android en Chrome OS zijn beide Linux.

Linux

Linux (ook wel GNU/Linux) is een veilige OS (Operation System) voor pc's (personal computers). Linux is de meest populaire OS op de wereld (android is ook linux) en is veel veiliger en betrouwbaarder dan windows of macos. Er zijn verschillende linux versies, ook wel bekent als distrobutions of distros, maar ze werken bijna allemaal precies hetzelfde. Er zijn 3 "main" distros: Red Hat/Fedora, Ubuntu/Debian en Arch. Bijna elke distro ooit is hier op gebaseert. Er zijn ook veel "independent" distros, zoals void of gentoo die op niks gebaseert zijn. Chrome OS is dan weer gebaseert op gentoo, Linux Mint (wat Olav en Tom hebben) is gebaseert op Ubuntu, wat weer gebaseert is op Debian. In theorie zijn "independent" distros beter, omdat ze niet op iets gebaseert zijn. Je kan linux ook "dual booten" en dit houdt in dat je op een andere disk of partition linux NAAST een andere OS installeert, bijvoorbeeld windows. Arch is "bleeding edge"/unstable (onstabiel (Loïs)) oftewel dat het heel snel nieuwe programma's (packages) krijgt. Debian krijgt juist geen tot bijna geen updates en is dus heel stabiel (stable). Dit klinkt als een slecht iets, maar omdat het juist geen updates krijgt breekt het dus ook bijna nooit. Arch daar in tegen breek heel snel. Fedora/OpenSUSE of void zitten daar precies tussenin. Het krijgt snel updates en packages en is toch erg stabiel.

Linux heeft verschillende "desktop environments", de meest bekende zijn gnome, kde plasma en xfce. Er zijn er natuurlijk veel meer, zoals lxqt, lxde, cinnamon, mate, budgie etc. Een desktop environment is hoe je distro eruit ziet. Een desktop environment kan je op elke distro installeren, dus je moet niet de distro beoordelen op hoe die eruit ziet. Je kan elke distro EXACT HETZELFDE maken. Het enige waar je naar moet kijken bij een distro is hoe je het installeert, wat de update cycle is (stable als debian, "leadinge edge" als fedora en opensuse of bleeding edge als arch, gentoo en void. Je hebt ook "window manager" en als de naam zegt managed het jou windows. Als je niet snapt wat ik hier mee bedoel, dit is letterlijk gewoon de rand om het programma zodat je hem kunt verplaatsen of full screen kan maken. Window managers zijn verplicht en elke desktop environment heeft zijn eigen window manager. Window managers zijn niet aan te raden voor beginners, maar je kan alles zelf instellen en het is extreem licht. Daarnaast telt ook de package manager mee voor het kiezen van je distro. Debian gebruikt apt (advanced packaging tool), fedora/redhat dnf en arch pacman (niet de game lol). Apt is oud en is de langzaamste, terwijl pacman een van de snelste is samen met xbps (void linux). Dnf zit daar weer tussen in.

FreeBSD

FreeBSD is ook een OS en lijkt sterk op linux. FreeBSDs code is alleen veel schoner en het is normaal gesproken ook veel sneller. Hardware support is alleen slecht, dus de kans dat dit werkt op jou pc/laptop is erg klein. Daarnaast is het ook meer bedoelt voor servers, omdat het veel stabieler is dan linux. De support van programma's is ook niet al te best (niet veel apps werken).

Android-x86

Android-x86 is een android OS voor pc's/laptops. Je kan alle apps installeren die je ook gewoon op je telefoon zou kunnen installeren. Het heeft ook de app store (of f-droid en aurora store als je dat fijner vindt) en is gewoon android. Zie ook Bliss OS, een fork van android-x86. Gebruik geen bluestacks, nox of memu android emulators. Ze zijn allemaal onbetrouwbaar en tyfus langzaam. Je kan wel android studio gebruiken om een hele android telefoon te runnen in windows of linux.

BalenaEtcher, Rufus en Ventoy

BalenaEtcher, Rufus en Ventoy zijn allemaal manieren om een ISO file te flashen naar je USB stick. Dit is nodig om bijvoorbeeld Windows of Linux te installeren. BalenaEtcher is de makkelijkste, maar gebruikt veel opslag en heeft weinig opties. Rufus iets gecompliceerder, maar geeft je meer opties en is een heel stuk lichter (1,3mb). Ventoy is alleen als je vele distro wilt uittesten EN genoeg opslag hebt op je usb stick (meer dan 32gb is aan te raden). Meestal wordt dit gebruikt om linux te installeren, maar het kan ook worden gebruikt voor windows. Linux heeft er ook een, genaamt dd.
