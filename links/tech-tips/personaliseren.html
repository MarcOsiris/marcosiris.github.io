Personaliseren

Dit zijn tips om hoe je je OS kan personaliseren.

PowerToys (FOSS)

PowerToys is gemaakt door Microsoft en is bedoelt om meer functies te geven voor Windows. Het staat ook op de Microsoft Store.

Lively Wallpaper (FOSS)

Lively Wallpaper is net als Wallpaper Engine een programma om een bewegende achtergrond te krijgen. Het heeft een paar standaard achtergronden, maar je kan zelf ook gifjes of videos toevoegen. Pas wel op, dit kan impact hebben op je performance want het gebruikt je GPU en CPU (zelfde geld voor Wallpaper Engine). Lively Wallpaper is wel geoptimaliseerd voor performance.

TranslucentTB (FOSS) en RoundedTB (FOSS)

TranslucentTB (ook wel TranslucentTaskBar) is, zoals de naam al zegt, een manier hoe je je taskbar doorzichtig kan maken. TranslucentTB kan je samen met RoundedTB gebruiken. RoundedTB zorgt ervoor dat je taskbar afgeronde hoeken heeft.

Rainmeter (FOSS)

Met Rainmeter kan je je desktop helemaal veranderen van hoe jij wilt. Je krijgt meer vrijheid en keuzes. Net als Lively Wallpaper gebruikt dit ook je GPU en CPU.

KISS Launcher (FOSS)

KISS Launcher is een launcher voor Android. KISS staat voor "Keep it Simple, Stupid". Een launcher is iets waarmee je je Android telefoon er anders uit kan laten zien. Het is licht, verbruikt minder batterij en is snel. Misschien moet je het wel een beetje aanpassen of eraan wennen. Ik gebruik het nu 2 dagen voor mijn telefoon en het is prettig te gebruiken (update: gebruik het al bijna een 3 maanden en heb nogsteeds geen problemen). KISS Launcher staat op F-Droid.

Delta (FOSS)

Delta is een icon pack voor Android. Kan worden gebruikt met KISS Launcher en het staat op F-Droid. Delta heeft ook achtergronden.
