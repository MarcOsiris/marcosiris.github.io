Films

Ik ga je vertellen hoe je (illegaal) films kan kijken.

Stremio

Stremio is een streaming service (zoals Netflix) en is ook (gedeeltelijk) FOSS. Het is makkelijk te gebruiken en te installeren. Je hebt wel "plugins" nodig om links te kunnen krijgen om te streamen, bijvoorbeeld van Torrentio.

VLC

VLC is een simpele multimedia player en kan videos en films afspelen. Erg aan te raden in plaats van de standaard windows player. Daarbij is het ook FOSS.

MPV

Net als VLC is MPV een multimedia player. Dit is zeer aan te raden met Stremio. Met VLC kan je gewoon rechter muizen klikken bij stremio (als je de film aan het kijken bent), terwijl het met MPV iets gecompliceerder is. Je moet naar je start menu van windows, type "stremio" en klik op "open file location". Rechtermuis klikie op de stremio shortcut en klik nog een keer op "open file location". Tussen die lijst staat "server.js". Open die met notepad(++) of een andere tekst editor. Klik "ctrl + f" (dat is geen knop, dat zijn physieke toetsen op je toetsenbord) en type "mpv". Klik "find next" totdat je het volgende vind:

 

mpv: {

title: "MPV",

args: ['--no-terminal'],

subArg: '--sub-file=', // DOESN'T WORK ON MAC

timeArg: '--start=',

playArg: '',

darwin: {

path: ["/usr/local/bin/mpv", "/opt/local/bin/mpv", "/sw/bin/mpv"] // Homebrew, MacPort, Fink

},

linux: {

path: ["/usr/bin/mpv"]

},

win32: {

path: ["C:/ProgramData/chocolatey/lib/mpv.install/tools/mpv.exe"]

}

 

Bij het stukje "win32:" moet je net als ik je path aangeven van waar mpv staat. VLC doet dit automatisch maar MPV niet. Ik heb het via chocolatey geinstalleerd, dus ik heb de path van chocolatey aangegeven. Als je dat hebt gedaan, moet je hem eerst opslaan en dan ben je klaar! Als je ondertiteling wilt gebruiken, moet je die eerst downloaden. Klik gewoon rechter muisknop als je de film aan het kijken bent in stremio en klik dan op "download subtitles".

Kodi

Kodi is net als stremio een manier om film af te spelen. Het dient meer als een OS en kan geen videos streamen, je moet ze eerst downloaden. Ik heb dit thuis op mijn tv en mijn vader gebruikt het vaak. Het kan zelfs RetroArch (zie "Emulators") gebruiken om te gamen! Stremio is meer aan te raden als je gewoon films wilt kijken.
