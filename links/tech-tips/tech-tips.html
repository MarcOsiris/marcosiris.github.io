Hoe weet ik of het wel betrouwbaar is?

Bijna elk programma hier is FOSS (Free and Open Source Software) en je kan dus de in de code (meestal op github of gitlab) controleren of het werkelijk betrouwbaar is. Je kan het ook opzoeken op reddit of aan mij vragen.

Waar zie ik de programma's?

Rechtboven bij het oranje vlak kan je alle kopjes zien. Elk programma staat bij het kopje waar het programma hoort.

Kan ik nog terug?

Natuurlijk kan je dat: terug naar de start!
