File Managers

Een file manager is een manier hoe je naar je files kan kijken. Dit zijn bijvoorbeeld je downloads, documenten, videos of foto's. De meest bekende is Verkenner (File Explorer op zijn Engels) die bij welke Windows installatie komt. Deze is vaak langzaam of onhandig, daarom zijn hier een paar alternatieven. Gebruik geen WinRAR of WinZip, voor deze moet je betalen en ze zijn closed source. Ook is 7-Zip VEEL beter dan beide.

7-Zip (FOSS)

De meest betrouwbare en snelste is 7-Zip. 7-Zip gebruikt .7z files. Heeft ook een Linux versie. 7-Zip kan ook worden gebruikt in verkenner. Als hij gedownload is, moet je het volgende doen: bovenaan staat tools > options > klik 1x op het +je van de LINKER kolom (tot dat ze allemaal 7-zip zijn) > Apply > OK. Nu kan je hem gebruiken in verkenner.

Simple File Manager Pro (FOSS)

Dit is gemaakt door Simple Mobile Tools en is bedoeld voor Android. Om foto's te bekijken zal ik Simple Gallery Pro aanraden (zie Android Apps).
